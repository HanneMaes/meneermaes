Je 1e Website Stylen

Style je vorige website door gebruik te maken van CSS

  1. Je start van een website met meerdere pagina’s en styled alle pagina’s met hetzelfde CSS-bestand.
  2. Je gebruikt enkel CSS in het .CSS-bestand en niet in je .html-bestand .
  3. Zorg ervoor dat je CSS-code een goede structuur heeft.
    Je maakt bijvoorbeeld gebruik van h1, h2, a als elementen dezelfde opmaak moeten hebben.
  4. Nadien geef je 1 pagina een ander ontwerp, door gebruikt te maken van een ander CSS-bestand.
  5. Ga zelfstandig opzoek naar nieuwe CSS-attributen om je website zo mooi mogelijk te maken.
Puntenverdeling: Je eerste website stylen
Elke pagina heeft CSS-styling 3
Je CSS-bestand heeft de correcte naam 1
Je geeft minstens 1 pagina een ander design door gebruik te maken van een ander CSS-bestand 2
Je CSS heeft een goede structuur 3
Je hebt enkel CSS-code in het .CSS-bestand 1
Hoe uitgebreid is je styling 5

Uitdagingen

  • Geef je pagina een afbeelding als achtergrond.
  • Geef een HTML-element een rand, geef deze rand ook een kleur en dikte.
  • Geeft een HTML-element een schaduw, geef deze schaduw ook een kleur.
  • Geeft een tekst een schaduw, geef deze tekst ook een kleur.
  • Maak elementen doorschijnend.
  • Surf naar https://fonts.google.com/ en gebruik lettertypes.
