Geef je gallery een **responsive menu**:
- Desktop: 
    - Een **horizontale** navbar.
- Mobile: 
    - Een **hamburger menu** dat een **full-screen menu** opent.

**Tip:** Met deze CSS-regel kan je een HTML-element verwijderen:

```css
.hide {
    display: none;
}
```
