# Wat doen deze HTML-tags?
- `<h1></h1>`, `<h2></h2>`, ..., `<h6>`
- `<small></small>`
- `<sup></sup>`
- `<center></center>`
- `<hr>`

# Opdracht: Zoek zelf HTML-tags

Start met deze code:
```html 
<p>
    E = MC2
    <br>
    H2O
</p>

<p>
    Wanneer ik hier met mijn muis over ga, komt er extra info tevoorschijn.
</p>
```
